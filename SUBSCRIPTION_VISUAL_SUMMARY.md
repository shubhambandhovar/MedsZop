## 🔗 API Quick Reference

```
GET  /api/subscription/plans                      Get plans
POST /api/subscription/plans                      Create plan (admin)

GET    /api/subscription/user/:userId             Get subscription
POST   /api/subscription/create                   Create subscription
PUT    /api/subscription/:id/medicines            Update medicines
PUT    /api/subscription/:id/pause                Pause
PUT    /api/subscription/:id/resume               Resume
PUT    /api/subscription/:id/skip-month           Skip
PUT    /api/subscription/:id/cancel               Cancel

POST   /api/consultation/create                   Book consultation
GET    /api/consultation/user/:userId             Get consultations
PUT    /api/consultation/:id/complete             Mark completed
PUT    /api/consultation/:id/cancel               Cancel

POST /api/subscription/cron/process-billings     Run billing (cron)
```
